---
Grab a friend for a competitive game of snake. Don't let them slither in front of you!
---
Software requirements:
	Python 3.10
	Raylib: install with py -m pip install raylib
---
Directory:
root				(project root folder)
- cycle				(source code for game)
	- game			(specific game classes)
		- casting	(acting objects)
		- directing	(director)
		- scripting	(actions for actors to take)
		- services	(input/output)
		- shared	(shared classes)
	- __main__.py		(game entry point)
	- constants.py		(values utilized by classes)
- README.txt
